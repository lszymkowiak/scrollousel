Autoplay can be set:

  • via data-* attribute 
    <div class="scrollusel" data-autoplay='[2000, "right", true]'>...</div>
  • programatically via options 
    $('.scrollusel').scrollusel({
    autoplay: [2000, "right", true]
});


Option name: autoplay

Value type: array

Format: [timeout, direction, pause-on-hover]

  • timeout (integer) - time in miliseconds
  • direction (string) - avaliable values "left", "right
  • pause-on-hover (boolean) - whether pause when mouse is over scrollousel

Default value: [0, "left", true]
