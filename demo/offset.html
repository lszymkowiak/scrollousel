Active slide left offset can be set:

  • via data-* attribute 
    <div class="scrollusel" data-offset='100'>...</div>
  • programatically via options 
    $('.scrollusel').scrollusel({
    offset: 100
});


Option name: offset

Value type: integer/string

Default value: 0

Available values:

  • integer - fixed offset in pixels
  • X% - percentage offset of container width
