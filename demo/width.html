Slide width can be set:

  • via data-* attribute 
    <div class="scrollusel" data-width='full'>...</div>
  • programatically via options 
    $('.scrollusel').scrollusel({
    width: 'full'
});


Option name: width

Value type: string/integer

Default value: 'auto'

Available values:

  • 'auto' - auto width, best suited for slides containing images
  • 'full' - slide width is inherited from container width
  • integer (integer) - fixed width in pixels
