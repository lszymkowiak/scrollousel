Start slide can be set:

  • via data-* attribute 
    <div class="scrollusel" data-start=':eq(3)'>...</div>
  • programatically via options 
    $('.scrollusel').scrollusel({
    width: ':eq(3)'
});


Option name: start

Value type: jQuery selector

Default value: :first

Available values: jQuery selectors
