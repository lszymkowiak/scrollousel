Active slide align can be set:

  • via data-* attribute 
    <div class="scrollusel" data-align='center'>...</div>
  • programatically via options 
    $('.scrollusel').scrollusel({
    align: 'center'
});


Option name: align

Value type: string

Default value: 'left'

Available values: 'left', 'center', 'right'
